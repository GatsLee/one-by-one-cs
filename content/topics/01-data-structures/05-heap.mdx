## 핵심 개념

### 힙(Heap)이란?

힙은 **완전 이진 트리** 형태의 자료구조로, 부모-자식 간에 특정 대소 관계를 유지한다.

> **비유**: 힙은 "회사 조직도"와 비슷하다. 최소 힙은 "직급이 낮을수록 위에 있는 조직도", 최대 힙은 "직급이 높을수록 위에 있는 조직도"다.

### 힙의 종류

**최소 힙(Min Heap)**: 부모 &lt;= 자식 (루트가 최솟값)
```
        1
       / \
      3   5
     / \ / \
    7  9 8  10
```

**최대 힙(Max Heap)**: 부모 >= 자식 (루트가 최댓값)
```
        10
       /  \
      9    5
     / \  / \
    7  3 1   4
```

### 힙의 핵심 성질

1. **완전 이진 트리**: 마지막 레벨을 제외하고 모든 레벨이 채워져 있고, 마지막 레벨은 왼쪽부터 채워진다.
2. **힙 순서 성질**: 최소 힙이면 부모 &lt;= 자식, 최대 힙이면 부모 &gt;= 자식.
3. **형제 간에는 대소 관계가 없다**: 같은 레벨의 노드들 사이에는 정렬 순서가 보장되지 않는다.

### 배열 기반 구현

완전 이진 트리는 배열로 효율적으로 표현할 수 있다. 인덱스 0부터 시작할 때:

```
트리:         1
             / \
            3   5
           / \ / \
          7  9 8  10

배열: [1, 3, 5, 7, 9, 8, 10]
인덱스: 0  1  2  3  4  5  6
```

| 관계 | 인덱스 공식 (0-based) |
|------|----------------------|
| 부모 | (i - 1) // 2 |
| 왼쪽 자식 | 2 * i + 1 |
| 오른쪽 자식 | 2 * i + 2 |

예시: 인덱스 1의 노드(값 3)
- 부모: (1-1)//2 = 0 → 값 1
- 왼쪽 자식: 2*1+1 = 3 → 값 7
- 오른쪽 자식: 2*1+2 = 4 → 값 9

### heappush (삽입) — Sift Up

새 원소를 힙에 추가하는 과정:

1. 배열의 **맨 끝**에 새 원소를 추가한다 (완전 이진 트리 유지).
2. 부모와 비교하여 힙 성질이 깨지면 **부모와 교환** (sift up).
3. 루트까지 올라가거나 힙 성질이 만족될 때까지 반복.

```
최소 힙에 2 삽입:

Step 1: 맨 끝에 추가        Step 2: 부모(9)와 비교    Step 3: 부모(3)와 비교
        1                           1                       1
       / \                         / \                     / \
      3   5                       3   5                   2   5
     / \ / \                     / \ / \                 / \ / \
    7  9 8  10                  7  2 8  10              7  3 8  10
          ↑                       ↑
       2 추가                  2<9 교환                 2<3 교환

Step 4: 부모(1)와 비교 → 2>1이므로 중단. 완료!
```

시간 복잡도: **O(log n)** (최대 트리 높이만큼 올라감)

### heappop (삭제) — Sift Down

최솟값(또는 최댓값)을 제거하는 과정:

1. 루트(최솟값/최댓값)를 제거한다.
2. 배열의 **맨 끝 원소**를 루트로 옮긴다.
3. 자식 중 더 작은(최소 힙) 자식과 비교하여 교환 (sift down).
4. 리프에 도달하거나 힙 성질이 만족될 때까지 반복.

```
최소 힙에서 최솟값(1) 제거:

Step 1: 루트 제거, 마지막 원소(10)를 루트로
        10
       /  \
      3    5
     / \  /
    7   9 8

Step 2: 자식(3, 5) 중 작은 3과 비교 → 10>3 교환
        3
       / \
     10   5
     / \ /
    7  9 8

Step 3: 자식(7, 9) 중 작은 7과 비교 → 10>7 교환
        3
       / \
      7   5
     / \ /
   10  9 8

Step 4: 리프 도달. 완료!
```

시간 복잡도: **O(log n)**

### 우선순위 큐(Priority Queue)

우선순위 큐는 **가장 높은 우선순위의 원소를 먼저 꺼내는** 추상 자료형이다. 힙은 우선순위 큐를 효율적으로 구현하는 자료구조다.

| 구현 방식 | 삽입 | 최솟값 삭제 | 최솟값 조회 |
|-----------|------|-----------|-----------|
| 정렬되지 않은 배열 | O(1) | O(n) | O(n) |
| 정렬된 배열 | O(n) | O(1) | O(1) |
| **힙** | **O(log n)** | **O(log n)** | **O(1)** |

### 힙의 대표 활용

1. **상위 K개 요소 찾기**: n개 중 가장 큰 K개를 찾으려면 크기 K의 최소 힙 사용
2. **중앙값 찾기**: 최대 힙(왼쪽 절반) + 최소 힙(오른쪽 절반) 사용
3. **다익스트라 최단 경로**: 가장 가까운 미방문 노드를 빠르게 선택
4. **작업 스케줄링**: 우선순위가 높은 작업을 먼저 처리

## 코드 예시

### 최소 힙 직접 구현

```python
class MinHeap:
    """배열 기반 최소 힙 구현"""

    def __init__(self):
        self.heap = []

    def _parent(self, i):
        """부모 인덱스 반환"""
        return (i - 1) // 2

    def _left_child(self, i):
        """왼쪽 자식 인덱스 반환"""
        return 2 * i + 1

    def _right_child(self, i):
        """오른쪽 자식 인덱스 반환"""
        return 2 * i + 2

    def _swap(self, i, j):
        """두 원소 교환"""
        self.heap[i], self.heap[j] = self.heap[j], self.heap[i]

    def push(self, val):
        """힙에 값 삽입 — O(log n)"""
        self.heap.append(val)              # 1. 맨 끝에 추가
        self._sift_up(len(self.heap) - 1)  # 2. 위로 올리며 정렬

    def _sift_up(self, i):
        """삽입된 원소를 올바른 위치로 올림"""
        while i > 0:
            parent = self._parent(i)
            if self.heap[i] < self.heap[parent]:  # 부모보다 작으면
                self._swap(i, parent)              # 교환
                i = parent                         # 부모 위치로 이동
            else:
                break                              # 힙 성질 만족 → 중단

    def pop(self):
        """최솟값 제거 및 반환 — O(log n)"""
        if not self.heap:
            raise IndexError("빈 힙에서 pop 불가")

        min_val = self.heap[0]                     # 1. 최솟값 저장
        last = self.heap.pop()                     # 2. 마지막 원소 꺼냄

        if self.heap:                              # 힙이 비지 않았으면
            self.heap[0] = last                    # 3. 마지막 원소를 루트로
            self._sift_down(0)                     # 4. 아래로 내리며 정렬

        return min_val

    def _sift_down(self, i):
        """루트로 올린 원소를 올바른 위치로 내림"""
        n = len(self.heap)
        while True:
            smallest = i
            left = self._left_child(i)
            right = self._right_child(i)

            # 왼쪽 자식이 더 작은지 확인
            if left < n and self.heap[left] < self.heap[smallest]:
                smallest = left
            # 오른쪽 자식이 더 작은지 확인
            if right < n and self.heap[right] < self.heap[smallest]:
                smallest = right

            if smallest != i:                      # 자식이 더 작으면
                self._swap(i, smallest)            # 교환
                i = smallest                       # 자식 위치로 이동
            else:
                break                              # 힙 성질 만족 → 중단

    def peek(self):
        """최솟값 조회 (제거 안 함) — O(1)"""
        if not self.heap:
            raise IndexError("빈 힙")
        return self.heap[0]

    def __len__(self):
        return len(self.heap)

    def __repr__(self):
        return f"MinHeap({self.heap})"


# === 사용 예시 ===
h = MinHeap()
for val in [5, 3, 8, 1, 7, 2]:
    h.push(val)
    print(f"push({val}): {h}")

print(f"\n최솟값: {h.peek()}")             # 1

while h.heap:
    print(f"pop → {h.pop()}")              # 1, 2, 3, 5, 7, 8 순서로 출력
```

### Python heapq 모듈 활용

```python
import heapq

# === 기본 사용법 ===
heap = []
heapq.heappush(heap, 5)
heapq.heappush(heap, 3)
heapq.heappush(heap, 8)
heapq.heappush(heap, 1)

print(heap)                    # [1, 3, 8, 5] — 내부 배열 형태

print(heapq.heappop(heap))    # 1 (최솟값)
print(heapq.heappop(heap))    # 3

# === 리스트를 힙으로 변환 — O(n) ===
data = [5, 3, 8, 1, 7, 2]
heapq.heapify(data)
print(data)                    # [1, 3, 2, 5, 7, 8]

# === 최대 힙 — 부호 반전 트릭 ===
max_heap = []
for val in [5, 3, 8, 1, 7]:
    heapq.heappush(max_heap, -val)     # 음수로 삽입

print(-heapq.heappop(max_heap))        # 8 (최댓값)
print(-heapq.heappop(max_heap))        # 7

# === nlargest / nsmallest ===
data = [5, 3, 8, 1, 7, 2, 9, 4]
print(heapq.nlargest(3, data))         # [9, 8, 7]
print(heapq.nsmallest(3, data))        # [1, 2, 3]
```

### 활용 1: 상위 K개 요소 찾기

```python
import heapq

def top_k_elements(nums, k):
    """
    n개의 숫자 중 가장 큰 k개를 찾기 — O(n log k)

    전략: 크기 k의 최소 힙을 유지.
    힙의 최솟값보다 큰 원소가 오면 교체.
    결과적으로 힙에는 가장 큰 k개만 남음.
    """
    min_heap = []

    for num in nums:
        if len(min_heap) < k:
            heapq.heappush(min_heap, num)
        elif num > min_heap[0]:            # 현재 최솟값보다 크면
            heapq.heapreplace(min_heap, num)  # pop + push 한번에

    return sorted(min_heap, reverse=True)  # 내림차순 정렬

print(top_k_elements([3, 1, 5, 12, 2, 11, 7], 3))  # [12, 11, 7]
```

### 활용 2: 스트리밍 중앙값 찾기

```python
import heapq

class MedianFinder:
    """
    스트리밍 데이터에서 실시간 중앙값을 찾기.

    전략:
    - max_heap: 작은 절반 (최대 힙으로, 부호 반전)
    - min_heap: 큰 절반 (최소 힙)
    - max_heap의 최댓값 <= min_heap의 최솟값 유지
    - 두 힙의 크기 차이는 최대 1
    """

    def __init__(self):
        self.max_heap = []   # 작은 절반 (음수로 저장)
        self.min_heap = []   # 큰 절반

    def add_num(self, num):
        # 1. 작은 절반(max_heap)에 먼저 추가
        heapq.heappush(self.max_heap, -num)

        # 2. max_heap의 최댓값이 min_heap의 최솟값보다 크면 이동
        if (self.min_heap and
            -self.max_heap[0] > self.min_heap[0]):
            val = -heapq.heappop(self.max_heap)
            heapq.heappush(self.min_heap, val)

        # 3. 크기 균형 맞추기 (max_heap이 최대 1개 더 많도록)
        if len(self.max_heap) > len(self.min_heap) + 1:
            val = -heapq.heappop(self.max_heap)
            heapq.heappush(self.min_heap, val)
        elif len(self.min_heap) > len(self.max_heap):
            val = heapq.heappop(self.min_heap)
            heapq.heappush(self.max_heap, -val)

    def find_median(self):
        if len(self.max_heap) > len(self.min_heap):
            return -self.max_heap[0]               # 홀수 개: max_heap의 최댓값
        return (-self.max_heap[0] + self.min_heap[0]) / 2  # 짝수 개: 두 중간값의 평균


# === 사용 예시 ===
mf = MedianFinder()
for num in [5, 3, 8, 1, 7]:
    mf.add_num(num)
    print(f"add({num}) → 중앙값: {mf.find_median()}")
# add(5) → 중앙값: 5
# add(3) → 중앙값: 4.0
# add(8) → 중앙값: 5
# add(1) → 중앙값: 4.0
# add(7) → 중앙값: 5
```

## 정리

| 항목 | 내용 |
|------|------|
| 힙 | 완전 이진 트리 + 부모-자식 대소 관계 유지 |
| 최소 힙 | 부모 &lt;= 자식, 루트가 최솟값 |
| 최대 힙 | 부모 >= 자식, 루트가 최댓값 |
| 배열 표현 | 부모: (i-1)//2, 왼쪽: 2i+1, 오른쪽: 2i+2 |
| heappush | 맨 끝에 추가 후 sift up — O(log n) |
| heappop | 루트 제거, 마지막 원소를 루트로 후 sift down — O(log n) |
| heapify | 배열을 힙으로 변환 — O(n) |
| Python heapq | 기본이 최소 힙, 최대 힙은 부호 반전으로 구현 |
| 우선순위 큐 | 힙으로 구현하면 삽입/삭제 O(log n), 조회 O(1) |
